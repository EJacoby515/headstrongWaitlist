TikTok Connection

One-time OAuth bootstrap for the HeadStrong brand TikTok account (@get.headstrong). Clicking Connect starts TikTok's authorization flow, returns an authorization code to our callback page, and our server exchanges that code for a long-lived refresh token used by the daily posting pipeline.

Saved locally in this browser. Use the sandbox key for testing, swap in the production key after review approval.

How this works:
1. This page generates a PKCE code_verifier + challenge and a random state.
2. You're redirected to tiktok.com/v2/auth/authorize/ where you approve the scopes.
3. TikTok sends you back to /tiktok-callback.html with a short-lived authorization code.
4. You copy the code into node scripts/tiktok-auth.js exchange "CODE|STATE" on the posting server, which exchanges it (with the client secret) for access + refresh tokens stored in our .env.

The access token refreshes automatically from then on. No user ever sees this page — it's only used by the HeadStrong team to authorize our own brand account.
